Udemy Clone – Online Learning Platform (HTML & CSS)

Developed a Udemy Clone using HTML5 and CSS3, replicating the design and user interface of a modern online learning platform. The project features a clean and user-friendly layout, including a navigation bar, course categories, featured courses section, learning recommendations, and a responsive footer. Focus was placed on creating an attractive visual design, proper content organization, and ensuring compatibility across different screen sizes using CSS styling and responsive design techniques.

Key Features:

navigation bar and menu
Course listing cards with images, ratings, and pricing
Featured and recommended courses sections
Clean and modern UI inspired by Udemy
Responsive design for desktop, tablet, and mobile devices
Structured layout using HTML5 semantic elements
Custom styling with CSS3, including flexbox

Technologies Used: HTML5, CSS3, Flexbox

This project helped strengthen front-end development skills, including webpage structuring, styling, layout design, and responsive web development.
